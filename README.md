spongebobify-racket
==

A port of [bencooper222/spongebobify](https://github.com/bencooper222/spongebobify) to [Racket](https://racket-lang.org/).

Usage
==

    > (require "spongebobify.rkt")
    > (spongebobify "hello world")
    "hElLo wOrLd"
    (spongebobify "hello world" #f)
    "HeLlO WoRlD"
